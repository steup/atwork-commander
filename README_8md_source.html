1  # atwork-commander
2 
3  [![Build Status](https://travis-ci.org/steup/atwork-commander.svg?branch=devel)](https://travis-ci.org/steup/atwork-commander)
4 
5  Complete reimplementation of the [old](https://github.com/robocup-at-work/at_work_central_factory_hub)
6  Referee Box (Refbox) for the @Work-League of RoboCup.
7 
8  This Refbox is a native ROS application.
9  However, it aims to enable multiple communication backends through individual plugins.
10  Additionally, a RViz GUI will be developed, which will provide enhanced visualization
11  capabilities for referees and visitors / spectators.
12  Generation of tasks conforming to the [rule book](https://github.com/robocup-at-work/rulebook)
13  are configurable using the ROS parameters in order to be easily adaptable for
14  any future changes in rules.

17  ## HowTo
18 
19  ### Starting the Refbox
20 
21  1. Start the __core__ and __com__ components:
22  ```
23  roslaunch atwork_commander atwork_commander.launch
24  ```
25  2. Generate a task using the CLI:
26  ```
27  roslaunch atwork_commander generate.launch task:=<task to generate>
28  ```
29  3. Wait for robots to register
30  4. Start the task execution using the CLI:
31  ```
32  roslaunch atwork_commander start.launch
33  ```
34 
35  For testing the Refbox without any robot a fake robot may be used using the **example_robot**:
36  `roslaunch atwork_commander example_robot.launch`
37 
38  ### Configurations
39 
40  - All launch files share the __refbox__ parameter to specify the Refbox to connect
41  to. Multiple Refboxes can be started on the same PC using multiple namespaces.
42  - When more information is needed, the __verbose__ parameter can be enabled for
43  more verbose logging.
44  - If a task needs to be sent to only some registered robots, the robots can be
45  specified in the __start__ command using the robots parameter in the format
46  "<team_name>/<robot_name>"
47 
48  ### Runtime control
49 
50  - the __forward__ launch file enables manual state change from *PREPARATION* to *EXECUTION*:
51  ```
52  roslaunch atwork_commander forward.launch
53  ```
54  - the __stop__ launch file enables stopping of the currently running task:
55  ```
56  roslaunch atwork_commander stop.launch
57  ```
